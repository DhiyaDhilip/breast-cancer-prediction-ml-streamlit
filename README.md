# breast-cancer-prediction-ml-streamlit
Classification project for predicting breast cancer diagnosis using Random Forest. Includes EDA, feature scaling, accuracy-based evaluation, and deployment as a Streamlit web app. The trained model is saved using Joblib for reuse.

#  Breast Cancer Prediction Using Machine Learning

##  Project Overview
This project predicts whether a breast tumor is malignant or benign using supervised machine learning techniques applied to medical diagnostic data.

##  Objectives
- Analyze diagnostic features
- Build a reliable classification model
- Provide real-time prediction through a web app

##  Tools & Technologies
- Python 3
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Joblib
- Streamlit

##  Workflow
1. Data Preprocessing and Feature Scaling  
2. Exploratory Data Analysis (EDA)  
3. Model Training using Random Forest Classifier  
4. Model Evaluation using Accuracy Score  
5. Model saving using Joblib  
6. Streamlit Deployment  

##  Model Performance
- Achieved high classification accuracy
- Model validated using accuracy metric
